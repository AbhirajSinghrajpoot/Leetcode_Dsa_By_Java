## 📂 Repository Architecture

The workspace is organized strictly by Data Structure and Algorithmic patterns to make navigation easy and logical.

| Directory | Core Concepts Covered | Key Algorithms / Patterns |
| :--- | :--- | :--- |
| `📁 Arrays` | Traversal, Manipulation, In-place operations | Two Pointers, Prefix Sum, Kadane’s Algorithm |
| `📁 BinarySearch` | Search space reduction, Matrix searching | Lower/Upper Bound, Search in Rotated Arrays |
| `📁 DP` | Optimal Substructure, Overlapping subproblems | Memoization, Tabulation, Knapsack, LCS |
| `📁 Heap` | Priority Queues, Min-Heap, Max-Heap | Top K Elements, K-way Merge |
| `📁 LinkedList` | Singly, Doubly, and Circular Lists | Fast & Slow Pointers (Floyd's Cycle Detection) |
| `📁 Queue and Stack` | LIFO / FIFO implementations | Monotonic Stack, BFS implementations |
| `📁 SlidingWindow` | Subarray and substring optimization | Fixed Window, Variable Window, Shrinking |
| `📁 Strings` | Parsing, Pattern Matching, Anagrams | Palindromes, ASCII manipulation, KMP (basics) |
| `📁 Tree` | Hierarchical data, BSTs, Traversals | DFS, BFS, Inorder, Preorder, Postorder |

*Note: The root directory contains `Main.java` and `Main1.java` which I use as scratchpads for testing code quickly before moving it to its dedicated folder.*

## 💻 How to Use and Run

If you want to clone and run this repository locally:

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/AbhirajSinghrajpoot/Leetcode_Dsa_By](https://github.com/AbhirajSinghrajpoot/Leetcode_Dsa_By)_...

```
 2. **Open in VS Code:**
   Ensure you have the "Extension Pack for Java" installed in VS Code.
 3. **Run a file:**
   Navigate to any .java file and click the Run button provided by the VS Code Java extension, or run via terminal:
   ```bash
   javac Main.java
   java Main
   
   ```
